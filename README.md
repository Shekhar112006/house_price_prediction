Project Overview

This is a Machine Learning project to predict house prices based on features like area, bedrooms, and bathrooms.
The project demonstrates Python modular coding, data preprocessing, model training, and prediction using Linear Regression.

Features

Clean modular structure:
data_preprocessing.py ‚Üí Load and split dataset
model_training.py ‚Üí Train and evaluate model
predict.py ‚Üí Make predictions
Uses Linear Regression from scikit-learn
Easy to extend with new algorithms
Fully GitHub-ready and professional

How to Run
Open notebook/main.ipynb in Jupyter Notebook.
Run all cells in order.The notebook will:
Load and split the dataset
Train the Linear Regression model
Evaluate the model
Predict house price for new inputs
